# Payment Gateway Standard

## Overview
Abstraction layer for multiple payment gateways supporting Flow, Global66, PayPal, Mercado Pago, and additional payment processors.

## Supported Gateways
- **Flow** - Chilean payment processor
- **Global66** - International payments
- **PayPal** - Global standard
- **Mercado Pago** - Latin America

## Features
- Unified interface across all gateways
- Error handling and retry logic
- Transaction logging
- Currency conversion support
- Webhook management
